Configurazione database
Inserisci le credenziali Supabase per connettere il CRM. Operazione una tantum per PC.
1
Vai su supabase.com → Settings → API → copia Project URL
2
Nella stessa pagina copia la Publishable key (inizia con sb_publishable_...)
3
Incolla i valori qui sotto e clicca Connetti
Le credenziali sono salvate solo sul tuo browser locale.
